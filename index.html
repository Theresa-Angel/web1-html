Introduction to HTML

  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page using a system of tags and attributes.
  • HTML consists of a series of elements.
  • HTML elements tell the browser how to display the content.
  • HTML documents are plain-text files saved with an .html extension.

    • A Simple HTML Document

    Example

    <!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

  <h1>My First Heading</h1>
  <p>My first paragraph.</p>

</body>
</html>

    Example Explained

    The <!DOCTYPE html> declaration defines that this document is an HTML5 document.

    The <html> element is the root element of an HTML page.

    The <head> element contains meta information about the HTML page.

    The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).

    The <body> element defines the document's body and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

    The <h1> element defines a large heading.

    The <p> element defines a paragraph.

    • 1991: HTML 1.0 (initial concept)
    • 1995: HTML 2.0 standardized by IETF
    • 1997‚Äì1999: HTML 3.2 and 4.01 by W3C
    • 2014: HTML5 finalized ‚Äî major upgrade with multimedia and semantic elements

    Features of HTML

    • Platform-independent
    • Supports multimedia (audio, video, images)
    • Allows linking via hyperlinks
    • Supports semantic structure (header, nav, article, etc.)
    • Works with CSS and JavaScript for styling and interactivity

    What is SGML?

  • SGML (Standard Generalized Markup Language) is a standard for defining generalized markup languages for documents.
  • HTML is an application of SGML.
  • SGML stands for Standard Generalized Markup Language.
  • It is derived from GML (Generalized Markup Language), which allowed users to work on standardized formatting styles for electronic documents.
  • It was developed and standardized by the International Organization for Standardization (ISO) in 1986.
  • SGML specifies the rules for tagging elements. These tags can then be interpreted to lay out document components in specific ways.

    • Use Case:

    SGML is used extensively to manipulate massive documents that undergo frequent revisions and need to be published in different formats. However, due to its complexity, it is not widely used on personal computers.

    Components of SGML

    • SGML provides a way to describe relationships between entities, elements, and attributes.
    • It defines how a computer can recognize the parts of a document using markup.
    • Rules help the computer identify where text elements start and end.
    • Document Type Definition (DTD) is used in SGML to define each element in a way a computer understands.

    Advantages of SGML

    • Highly customizable and extensible markup standard.
    • Suitable for large, complex documents and systems.
    • Separates content from presentation.
    • Platform-independent and widely adopted in publishing industries.

    Disadvantages of SGML

    • Very complex syntax compared to HTML or XML.
    • Steep learning curve for beginners.
    • Requires specialized parsers and tools.
    • Less suitable for lightweight or modern web development.

    HTML Tags

  • HTML Tags are fundamental elements used to structure and format content on web pages.
  • They provide instructions to web browsers on how to render text, images, links, and other media.
  • HTML tags are enclosed in angle brackets <> and usually come in pairs: an opening tag and a closing tag.
  • The closing tag has the same name as the opening tag, but includes a forward slash /.
  • The opening tag marks the beginning of an element, and the closing tag marks the end.

    • Example:

    <p>This is a paragraph.</p>

    Some tags are self-closing, meaning they don‚Äôt require a closing tag. Examples include:

    <img src="image.png" alt="Example Image">
<br>

    Essential tags for defining an HTML document structure include:

    • <!DOCTYPE html>
    • <html>
    • <head>
    • <body>

    Single (Self-closing) Tags

    • <br> - Line Break
    • <img> - Image
    • <hr> - Horizontal Rule
    • <input> - Input Field

    Dual Tags

    • <p>...</p> - Paragraph
    • <h1>...</h1> - Heading
    • <div>...</div> - Division
    • <a>...</a> - Anchor Link

    Head Section

  • The <head> section of an HTML document contains metadata and links to scripts, stylesheets, and other resources that are not directly displayed on the webpage.
  • It is placed before the <body> tag.

    • Common Tags Found in <head>:

    • <title> ‚Äì Sets the page title shown in the browser tab
    • <style> ‚Äì Defines internal CSS styles
    • <meta> ‚Äì Provides metadata like charset, description, author, etc.
    • <link> ‚Äì Links to external CSS or resources
    • <script> ‚Äì Embeds JavaScript or links to an external JS file
    • <base> ‚Äì Sets a base URL for relative links

    Metadata Tags Examples:

    • <meta charset="UTF-8">
    • <meta name="description" content="Intro to HTML">
    • <meta name="author" content="Your Name">
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>

    Description: Sets the page title shown in the browser tab and used by search engines.

    Syntax: 
    <title>Page Title</title>
    Example: 
    <title>My Website</title>

    Additional Info: Should be unique and descriptive for SEO purposes.

    <style>

    Description: Contains internal CSS to style the webpage.

    Syntax: 
    <style>
  selector {
    property: value;
  }
</style>
    Example: 
    <style>
  body {
    background-color: lightblue;
  }
</style>

    Additional Info: Use for quick styles. For large projects, prefer external stylesheets.

    <meta>

    Description: Provides metadata about the HTML document such as character encoding and description.

    Syntax: 
    <meta name="value" content="value">
    Example: 
    <meta charset="UTF-8">
<meta name="description" content="Learn HTML Basics">

    Additional Info: Important for search engine optimization and browser rendering.

    <link>

    Description: Links external files like stylesheets or icons to the document.

    Syntax: 
    <link rel="stylesheet" href="style.css">
    Example: 
    <link rel="stylesheet" href="main.css">

    Additional Info: Place all external stylesheets in the head for faster page load.

    <script>

    Description: Adds JavaScript to your HTML page either inline or via external file.

    Syntax: 
    <script src="script.js"></script>
    Example: 
    <script>
  alert('Hello World');
</script>

    Additional Info: Place at the end of the body for better performance unless essential in head.

    <base>

    Description: Sets the base URL for all relative links on the page.

    Syntax: 
    <base href="https://example.com/">
    Example: 
    <base href="https://mywebsite.com/">

    Additional Info: Only one <base> tag should be used per document, and it must be in the head.
